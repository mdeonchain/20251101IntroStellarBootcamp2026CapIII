# Todo List - Rama daniel-todo-cli

Esta rama contiene mi implementación del **Todo List** para el Bootcamp.

## Cambios principales

- Nueva funcionalidad de agregar, eliminar y marcar tareas como completadas.
- Interfaz CLI (`todo-cli`) para gestionar las tareas.
- Archivos modificados/creados:
  - `src/main.rs` (lógica principal)
  - `Cargo.toml` (dependencias si hubo cambios)
  - Otros archivos de soporte para el CLI.

## Cómo ejecutar

1. Clonar el fork y cambiar a la rama:

```bash
git clone https://github.com/daniel5312/20251101IntroStellarBootcamp2026CapIII.git
cd 20251101IntroStellarBootcamp2026CapIII
```
